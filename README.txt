AI SOLUTION
A Production-Ready Secure Authentication System
AI-SecureAuth is a robust user registration and login system built with Python (Flask). Designed for modern web security standards, it features a custom Baby Pink & White UI, Multi-Factor Authentication (MFA), and automated account protection. This project demonstrates a full development lifecycle—from secure coding to cloud optimization on Render.

✨ Product Highlights
Multi-Factor Auth: Integrated 2FA via Email OTP, Google OAuth 2.0, and reCAPTCHA v3 bot protection.

Account Hardening: Automated 5-attempt lockout, 90-day password expiry, and a strict non-reuse policy.

Role Management: Built-in Role-Based Access Control (RBAC) with dedicated Admin and User dashboard routes.

Audit-Ready: Integrated security logging system tracking all critical user actions and system changes.

Modern UX: Fully responsive "Baby Pink" themed interface optimized for both mobile and desktop browsers.

🏗️ Tech Stack
Backend: Python / Flask / SQLAlchemy (ORM)

Security: Bcrypt (Hashing), PyOTP (MFA), Flask-Talisman (Security Headers)

Deployment: Render Cloud, Gunicorn (WSGI), Python 3.11

Database: SQLite

⚙️ Local Setup
Clone the repository.

Install dependencies: pip install -r requirements.txt

Configure your .env file with your Google API and SMTP credentials.

Run the application: python app.py
